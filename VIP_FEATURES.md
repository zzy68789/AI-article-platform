# VIP 会员功能说明

## 会员套餐

| 类型 | 价格 | 配额 | 功能 |
|-----|------|------|------|
| 免费用户 | $0 | 5篇 | 基础功能 |
| **永久会员** | **$199** | **无限** | **全部功能** |

## VIP 专属权益

### 1. 无限创作配额 ✨
- 免费用户：每天 5 篇文章
- VIP 会员：**无限次创作**，随心所欲

### 2. AI 智能生图 🎨
- **Nano Banana AI**：使用 Gemini 生成独特创意配图
- **仅 VIP 可用**，免费用户无法使用

### 3. SVG 图表生成 📊
- **SVG Diagram**：自动生成精美的概念示意图和思维导图
- **仅 VIP 可用**，免费用户无法使用

### 4. AI 大纲智能编辑 ✍️
- 使用 AI 助手快速优化文章大纲结构
- **仅 VIP 可用**，免费用户只能手动编辑

### 5. 优先生成队列 ⚡
- VIP 会员享受更快的生成速度
- 优先处理，减少等待时间

### 6. 终身有效 💎
- 一次购买，永久使用
- 无需续费，享受所有未来更新

## 购买流程

1. 登录账号
2. 访问 `/vip` 页面
3. 点击 "立即购买"
4. 在 Stripe 支付页面完成支付
5. 自动升级为永久会员

## 退款政策

- ✅ 支持 7 天无理由退款
- ✅ 退款后自动取消会员权限
- ✅ 通过 API 或后台一键退款

## 技术实现

### 后端
- **支付集成**：Stripe Checkout Session
- **Webhook 处理**：幂等性保证
- **签名验证**：防止伪造请求
- **权限控制**：AOP + 注解

### 前端
- **VIP 标识**：导航栏显示黄金徽章
- **功能限制**：灰色禁用 + 提示升级
- **购买页面**：精美 UI + FAQ
- **支付跳转**：Stripe Hosted Page

### 数据库
- **user 表**：`isVip`、`vipTime`、`userRole`
- **payment_record 表**：支付记录、退款记录

## 代码位置

### 后端
- `PaymentService` - 支付服务
- `PaymentController` - 支付接口
- `StripeWebhookController` - Webhook 处理
- `QuotaServiceImpl` - 配额校验（VIP 无限）
- `ArticleServiceImpl` - 功能权限校验

### 前端
- `/pages/VipPage.vue` - 会员购买页面
- `/components/GlobalHeader.vue` - VIP 标识展示
- `/pages/article/ArticleCreatePage.vue` - 功能限制提示
- `/pages/article/components/OutlineEditingStage.vue` - AI 编辑限制
